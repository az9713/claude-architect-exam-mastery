# React Testing Conventions

Use React Testing Library.
Prefer userEvent over fireEvent.
Mock at module boundaries...

The YAML frontmatter paths field accepts glob patterns. Rules load only when editing matching files, reducing irrelevant context.

Glob Pattern Examples

  • **/*.test.tsx — All React test files, any directory
  • terraform/**/* — All Terraform files
  • src/api/**/*.ts — API TypeScript files only
  • packages/*/src/** — All monorepo package sources
  • **/*.{py,pyi} — All Python files
vs Subdirectory CLAUDE.md: Glob patterns apply to files spread across many directories. CLAUDE.md files are directory-bound. Use rules for cross-directory conventions.
